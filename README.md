# pencil_design

pencil*cursorでデザインの作成およびコードの生成をテストするリポジトリです。

## 漢方薬コーポレートサイト（Next.js）

- **要件**: Node.js 18.17 以上（推奨: 20.x）
- **開発**: `npm run dev` → http://localhost:3000
- **ビルド**: `npm run build` / **本番起動**: `npm run start`

### ページ

- `/` サイトトップ
- `/about` 会社概要
- `/products` 製品一覧
- `/history` 沿革
- `/contact` お問い合わせ
- `/sitemap` サイトマップ
